# Lil Poke

> ⚠️ **This project is still under active development.** Expect incomplete features, bugs, and occasional breaking changes as development progresses.

## About

Lil Poke is a free Pokémon fan game currently in active development. The goal is to deliver a fresh Pokémon experience while staying true to the spirit of the original games.

## Features

* Supports Pokémon from Generations 1–9
* Offers a fresh and unique Pokémon gameplay experience
* Completely free to play and will never be commercialized
